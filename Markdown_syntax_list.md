# Markdown_syntax_list

Text Formatting
-----------------
Bold: Wrap text in double asterisks (**bold text**) or double underscores (__bold text__).
Italic: Wrap text in single asterisks (*italic text*) or single underscores (_italic text_).
Bold and Italic: Wrap text in triple asterisks (***bold and italic text***) or triple underscores (___bold and italic text___).
Strikethrough: Wrap text in double tildes (~~strikethrough text~~).
Inline Code: Wrap text in single backticks (`inline code`).
